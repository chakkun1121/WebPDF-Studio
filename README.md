# WebPDF Studio

This is a free web application for editing PDFs in a web browser.

このアプリは web ブラウザで PDF を編集する無料アプリです。

## なぜ完全無料かつ広告なしで提供しているのか

- ファイルの処理をローカルで実行するためサーバーがいらないため
